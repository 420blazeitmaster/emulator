What is NESbox?

NESbox is an emulator of NES, Super Nintendo, Sega Mega Drive video consoles, built on Adobe Flash technology and it can only be run directly in your browser's window.

View on GitHub

To play this game, please, download the latest Flash player!

What ROM types does the emulator support?

The emulator supports .nes .smc .gen .gb .gbc .gba files.

What can I do on the website now?

If you have ROM files you can load them into the emulator and play.
